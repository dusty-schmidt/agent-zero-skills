# Design Suite

Unified toolkit for visual design, web interfaces, theming, and branding.

## Modules

### 1. Visual Art & Canvas
Create beautiful visual art in .png and .pdf formats.
- Posters, artwork, static designs
- Original visual designs
- Design philosophy and composition

### 2. Frontend Design  
Build production-grade frontend interfaces with high design quality.
- Web components, pages, applications
- Websites, landing pages, dashboards
- React components, HTML/CSS layouts
- Polished UI avoiding generic AI aesthetics

### 3. Theme Factory
Style artifacts with pre-set or custom themes.
- 10 pre-set themes with colors/fonts
- Apply to slides, docs, reports, HTML pages
- Generate custom themes on-the-fly

### 4. Brand Guidelines
Apply official brand standards to artifacts.
- Anthropic brand colors and typography
- Company design standards
- Visual formatting and consistency

## Use Case Matrix

| Request | Module |
|---------|--------|
| Create a poster | Canvas Design |
| Build a web page | Frontend Design |
| Style a presentation | Theme Factory |
| Apply brand colors | Brand Guidelines |
| Create art piece | Canvas Design |
| React component | Frontend Design |

## Design Principles
- Distinctive, production-grade quality
- Avoid generic AI aesthetics
- Creative, polished output
- Consistent with brand/theme when specified
