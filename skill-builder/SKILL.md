# Skill Builder

Unified toolkit for creating Agent Zero skills and MCP servers.

## Modules

### 1. Skill Creator
Create and optimize Agent Zero skills.
- Create skills from scratch
- Update and improve existing skills
- Run evaluations and benchmark performance
- Optimize skill descriptions for better triggering
- Measure skill performance with variance analysis

### 2. MCP Builder
Build Model Context Protocol servers.
- Create high-quality MCP servers
- Enable LLMs to interact with external services
- Well-designed tools for external APIs
- Support for Python (FastMCP) and Node/TypeScript (MCP SDK)

## Use Cases

| Task | Module |
|------|--------|
| Create new skill | Skill Creator |
| Improve existing skill | Skill Creator |
| Test skill performance | Skill Creator |
| Build API integration | MCP Builder |
| Create custom tool | MCP Builder |

## Workflow

### Skill Creation
1. Define use case and trigger conditions
2. Write SKILL.md with proper metadata
3. Add scripts and resources
4. Test with evaluation framework
5. Optimize description for accuracy

### MCP Server Creation
1. Identify external service/API
2. Design tool interfaces
3. Implement in FastMCP or MCP SDK
4. Test tool functionality
5. Document usage patterns
